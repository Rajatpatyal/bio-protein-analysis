 🔬 Bio Protein Analysis (BioAI Starter Project)

This project analyzes protein structures using Biopython.

## 🚀 Features
- Extract amino acid sequence (auto)
- Count residues and chains
- Compute amino acid frequency
- Analyze 3D coordinates
- Calculate structure geometry

## 🧪 Example Output
- Chains: 1  
- Residues: 312  
- Unique amino acids: 20  

## ⚙️ Setup

pip install -r requirements.txt

## ▶️ Run

python analyze_protein.py

## 📂 Dataset

Protein structure from AlphaFold database.
